# Frontend — AI Career Assistant

React + Vite frontend for the AI Career Assistant.

See the full documentation in the [root README](../README.md), including:
- Setup and configuration
- Local development instructions
- Production build and deployment
- Admin panel usage

## Quick start

```bash
npm install
npm run dev        # dev server at http://localhost:5173
npm run build      # production build → dist/
```

The dev server proxies `/api` requests to `http://localhost:8000` (backend).  
Make sure the backend is running before opening the app.
